Install instructions

Follow these steps to install the Listy web app:

  1. Finish reading these instructions and tap "Go to install page".
  2. On the bottom bar in Safari, tap the blue box with the arrow coming out of it.
  3. Scroll through the options in the lower list until you see "Add to Home Screen".
  4. In the dialog that pops up next, tap "Add".
  5. Go to your home screen.
  6. Tap the "Listy" app wiith the checkmark icon, and you're done! Enjoy the Listy app.
